这个文件是放在 GitHub 仓库内查看的项目数据库入口，用来集中追踪 IDRINK MVP 的仓库、Notion、Issue、PR 和任务。

## Current Scope

| Area | Status | Notes |
| --- | --- | --- |
| Profile | In progress | Personal tolerance profile |
| Body status | In progress | Empty stomach, sleep, fatigue, medication, unwell state |
| Drink record | In progress | Type, ABV, volume, quantity, speed |
| Risk calculation | In progress | Rule-based local calculation |
| Advice | In progress | Rule-based templates, no real AI API |
| Session ending | In progress | End current drinking session locally |

## Tracking Rules

| Type | Use |
| --- | --- |
| Repository | Project-level entry and documentation |
| Issue | Bugs, feature requests, MVP follow-ups |
| Pull Request | Code review and merge tracking |
| Task | Local or planning-only work that does not need a GitHub Issue yet |

## MVP Guardrails

- Keep native WeChat Mini Program structure.
- Do not add backend services in the first MVP phase.
- Do not connect a real AI API in the first MVP phase.
- Do not add drinking history lists unless explicitly requested.
