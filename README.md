# IDRINK MVP

IDRINK 是一个原生微信小程序 MVP，用于在酒局中记录饮酒、填写身体状态、计算风险，并给出规则化饮酒建议。

## 当前阶段

- 已创建 5 个核心页面：首页、个人酒量档案、身体状态、记一杯、饮酒建议
- 使用本地存储保存档案、身体状态、当前酒局和本场饮酒记录
- 使用规则计算纯酒精摄入、标准杯、风险等级和建议话术
- 不引入后端、不接真实 AI、不做历史列表
