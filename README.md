# IDRINK MVP

IDRINK 是一个原生微信小程序 MVP，用于在酒局中记录饮酒、填写身体状态、计算风险，并给出规则化饮酒建议。

## GitHub 内项目数据库

这个仓库内的项目数据库入口：

- [docs/github-database.md](docs/github-database.md)

它用于集中查看 IDRINK 的仓库信息、Notion 数据库入口、当前 MVP 范围和后续 Issue/PR 管理方式。

## 外部入口

- Notion GitHub 数据库：https://app.notion.com/p/f1832d36832c4c59a7d0c2328a7d3ea0
- IDRINK Notion 项目记录：https://app.notion.com/p/39025eb49d7e81439fc8f1017c0c6300

## 当前 MVP 范围

- 建档
- 填身体状态
- 记录饮酒
- 计算风险
- 查看饮酒建议
- 结束酒局

暂不引入后端、不接真实 AI、不做历史列表。
